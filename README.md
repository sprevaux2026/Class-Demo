# Class-Demo

This folder contains a small Flappy Bird–style web game: "Bull Flappy".

Files added:

- `index.html` — game page
- `style.css` — basic styles
- `main.js` — game logic (canvas)
- `assets/bull.svg` — bull image used as the player
- `assets/usf.svg` — USF logo used as collectible coins

Run:

1. Open `index.html` in a browser (double-click or `open`), or run a simple HTTP server:

```bash
cd Class-Demo
python3 -m http.server 8000
# then open http://localhost:8000 in a browser
```

Controls:
- Click or press Space to flap / start.
- Collect USF logos for points. Click to restart after game over.

Enjoy!
